## Project Overview
This project is an art museum created by machine learning artificial intelligence.  This museum appears in form of machine learning art.  The aim of this project is to investigate into ways in which human emotional can be depicted, challenged, and re-formed in the time of artificial intelligence. 

Every piece in the museum has a corresponding mental state,  like happiness, loneliness, worry, love, and hope. The museum collectively leads audiences to think about ways in which a.
